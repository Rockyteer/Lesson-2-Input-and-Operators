## Operators
### Arithmetic Operators

Used to perform mathematical operations
1. Addition `+`
2. Subtraction `-`
3. Multiplication `*`
4. Division `/` (Exact Answer)  `//` (Quotient Division)
5. Modulo `%` (remainder)
6. Exponent `**`

Ex.
```python # Code Block
print(5 / 2) #2.5
print(5 // 2) #2
print(5 % 2) #1
print(4 ** 2) #16
```

### Assignment Operators

1. `=` Simple Assignment
2. `+=` Addition Assignment
3. `-=` Subtraction Assignment
4. `*=`, `/=`, `//=`, `**=`, `%=`

Ex
```python
a = 2
a += 3 #5, a = a + 3
a = 1 #1
a = 3 #3
```


### Comparison Operators
1. `==` equal to
2.  `>=` greater than or equal to
3.  `>` greater than
4.  `<=` less than or equal to
5.  `<` less than
6.  `!=` not equal to


### Boolean operators
1. and (Precedence)
2. or
3. not
